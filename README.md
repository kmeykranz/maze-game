# ⚡ 不走迷宫就出不去的房间

一个纯前端单文件迷宫小游戏，主打霓虹赛博视觉、逐级扩大的迷宫挑战，以及带有策略性的道具系统。

你需要从起点出发，穿过随机生成迷宫，到达右下角终点。每次通关后迷宫尺寸会增加，难度持续上升。

## 预览特性

- 随机迷宫生成（并查集 + 随机边）
- 逐关升级：`14x14` 起步，每关 `+2`，最高挑战到 `24x24`
- 多种道具系统：钥匙、护盾、加速、传送、指南针
- 实时计时器与状态栏
- 键盘 + 按钮双操作方式
- 赛博风动态背景、粒子通关特效、发光 UI

## 游戏目标

1. 从左上角起点出发。
2. 移动到右下角终点（🏁）。
3. 通关后进入更大迷宫。
4. 当尺寸超过 `24` 时触发毕业结局。

## 操作说明

### 移动

- 方向键：`↑ ↓ ← →`
- 或：`W A S D`

### 功能键

- `空格`：智能使用道具（优先尝试开墙钥匙，其次护盾、加速）

### 面板按钮

- `⟳ 重启`：完整重置（回到 14x14，清空道具和计时）
- `▶ 开始`：开始/继续计时与游戏
- `⏸ 暂停`：暂停移动与计时
- `🔑 用钥匙`：尝试打开周围一面墙
- `🛡️ 护盾`：激活一次撞墙破壁保护
- `⚡ 加速`：激活连续移动增益

## 道具系统

游戏中会在部分格子随机生成道具（默认约占总格子数 `5%`）：

- `🔑 KEY`：消耗 1 把钥匙，打通一面相邻墙壁
- `🛡️ SHIELD`：激活后，下一次撞墙会直接破墙并消耗护盾
- `⚡ SPEED`：激活后可获得连续移动次数（当前实现为 2 次）
- `🌀 TELEPORT`：拾取后立即随机传送到迷宫内其他位置（不传到终点）
- `🧭 COMPASS`：短时间显示从当前位置到终点的路径提示

## 关卡与计时

- 计时器精度为 `50ms` 刷新。
- 每次通关会播放粒子特效，并在短暂过场后进入下一关。
- 完整重启会清空计时、回到初始尺寸。

## 运行方式

这是一个无依赖静态页面项目，直接打开即可运行：

1. 双击 `index.html` 在浏览器打开。
2. 或使用任意静态服务器启动后访问。
